# Plan 7 — v0.3.0-alpha.3: GGUF Q4_0 format reader

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Load llama.cpp-style GGUF Q4_0 checkpoints directly. Adds a native `LinearWeight::Q4Gguf` variant that respects GGUF's exact block layout (32 weights / 18-byte block: f16 scale + 16 packed bytes with biased nibbles, low-nibbles-first-half / high-nibbles-second-half), plus a minimal GGUF file parser that walks header + metadata + tensor descriptors and pulls out the tensors we need.

**Architecture:** New crate-internal module `gguf` in `ai-engine-runtime` covering: file header / metadata parsing (key-value pairs, primitive + string + array types), tensor descriptor parsing (name, dimensions, ggml_type, offset), and tensor data extraction (mmap'd slice for each tensor). A new `Q4GgufTensor<B>` mirrors `Q4Tensor<B>` but with GGUF's native block layout — 32-weight blocks, biased nibbles (`stored - 8 = signed`), low-half-low-nibble + high-half-high-nibble convention, f16 per-block scale. `LinearWeight::Q4Gguf(Q4GgufTensor<B>)` is added alongside the existing variants. A GGUF→ai-engine name translator handles the standard `blk.N.attn_q.weight` → `model.layers.N.self_attn.q_proj.weight` mapping. A new `load_gguf` entry point in `loader.rs` builds `LoadedWeights<B>` from a `.gguf` file, dispatching on each tensor's ggml_type.

**Tech Stack:** No new external crates. Reuses `memmap2`, `bytemuck`, `half::f16`. The GGUF format is well-specified at https://github.com/ggml-org/ggml/blob/master/docs/gguf.md — we implement just enough to load Q4_0 + F32 + F16 + BF16 tensors.

**Scope rule:** Plan 7 ships **GGUF Q4_0 only** (plus F32 / F16 / BF16 for boundary tensors like embedding + norms). Other ggml types (Q4_1, Q4_K, Q5_*, Q6_K, Q8_0, IQ_*) are explicitly NOT in this plan — each has its own bit-packing convention. Plan 8 expands.

**Baseline:** Branch `main` at `v0.3.0-alpha.2`. 175 tests + 4 ignored. Clippy clean.

## GGUF format reference (locked in here)

The GGUF file structure we target:

```
magic       : u32 = 0x46554747 ("GGUF" little-endian)
version     : u32 (we target v3; reject v1/v2)
tensor_cnt  : u64
metadata_cnt: u64

[metadata_cnt] key-value pairs:
  key  : gguf_string (u64 length + UTF-8 bytes)
  type : u32 (enum)
  value: depends on type

[tensor_cnt] tensor descriptors:
  name        : gguf_string
  n_dims      : u32
  shape       : [n_dims] u64
  ggml_type   : u32 (enum: F32=0, F16=1, Q4_0=2, ...)
  offset      : u64 (from start of tensor data section)

padding to align to `general.alignment` (default 32 bytes)

[tensor_cnt] tensor data blobs, in order
```

Value types we handle (others reject):
- `U8=0`, `I8=1`, `U16=2`, `I16=3`, `U32=4`, `I32=5`, `U64=10`, `I64=11`
- `F32=6`, `F64=12`
- `BOOL=7`
- `STRING=8` (u64 length + UTF-8)
- `ARRAY=9` (u32 element type + u64 count + flat array of values)

ggml_type values we handle:
- `F32 = 0`
- `F16 = 1`
- `Q4_0 = 2`
- `BF16 = 30` (some recent GGUFs use this; check actual value via metadata)

Q4_0 block layout (canonical from ggml.c):

```c
typedef struct {
    uint16_t d;          // delta (scale), stored as f16 (2 bytes)
    uint8_t  qs[16];     // 16 bytes = 32 nibbles
} block_q4_0;            // total 18 bytes per 32 weights

// Unpacking:
//   For block_q4_0 b, the 32 weights are recovered as:
//   for j in 0..16:
//     low_nibble  = b.qs[j] & 0x0F
//     high_nibble = (b.qs[j] >> 4) & 0x0F
//     signed_lo = (low_nibble  as i8) - 8  // gives -8..7
//     signed_hi = (high_nibble as i8) - 8
//     weight[j]      = signed_lo * f16_to_f32(b.d)   // FIRST half (indices 0..16)
//     weight[j + 16] = signed_hi * f16_to_f32(b.d)   // SECOND half (indices 16..32)
```

**Critical**: in GGUF Q4_0, byte `qs[j]`'s LOW nibble is the value at index `j` of the block, and the HIGH nibble is the value at index `j + 16`. This is DIFFERENT from our own Q4 format where low/high nibbles are columns 2k / 2k+1. The two layouts must be carefully distinguished in code.

**Per-block scale**: f16 stored in the block, applied to the full 32-weight block.

GGUF tensor name conventions (Llama-family):

```
token_embd.weight                      <-> model.embed_tokens.weight
output_norm.weight                     <-> model.norm.weight
output.weight                          <-> lm_head.weight (often absent if tied)
blk.{i}.attn_norm.weight               <-> model.layers.{i}.input_layernorm.weight
blk.{i}.attn_q.weight                  <-> model.layers.{i}.self_attn.q_proj.weight
blk.{i}.attn_k.weight                  <-> model.layers.{i}.self_attn.k_proj.weight
blk.{i}.attn_v.weight                  <-> model.layers.{i}.self_attn.v_proj.weight
blk.{i}.attn_output.weight             <-> model.layers.{i}.self_attn.o_proj.weight
blk.{i}.ffn_norm.weight                <-> model.layers.{i}.post_attention_layernorm.weight
blk.{i}.ffn_gate.weight                <-> model.layers.{i}.mlp.gate_proj.weight
blk.{i}.ffn_up.weight                  <-> model.layers.{i}.mlp.up_proj.weight
blk.{i}.ffn_down.weight                <-> model.layers.{i}.mlp.down_proj.weight
```

GGUF tensor shape order is **opposite of HF**: GGUF stores `[in, out]` while HF stores `[out, in]`. For our purposes, GGUF's [in, out] is **already math order** — we don't transpose Q4Gguf weights, matching how we treat our own Q4Tensor (pre-transposed).

## Important pre-flight notes

- **GGUF version 3** is the current stable. Reject v1/v2 with a clear error (their layouts differ in detail).
- **Endianness**: GGUF is little-endian. All multi-byte integers are little-endian; f16 follows IEEE 754 binary16 LE.
- **Alignment**: tensor data starts at an offset aligned to `general.alignment` (metadata key, default 32). The file header gives offsets *relative to the tensor data section start*, NOT from file start.
- **TinyLlama / SmolLM are too big for a fixture.** We don't ship a downloaded GGUF in the repo. Plan 7's fixture is generated by converting our existing `toy-llama-3` bf16 safetensors → HF format → llama.cpp's `convert_hf_to_gguf.py` → quantize to Q4_0. The fixture-generation script orchestrates that pipeline.
- **The reference_logits.bin** for the GGUF fixture is the SAME bf16 reference as `toy-llama-3`. GGUF Q4_0 forward error is intrinsically high on random weights (similar phenomenon as our own Q4 — top-10 logits span < Q4 noise), but the dispatch-parity test (GGUF Q4_0 forward bit-identical to dequant-then-Dense) is what proves correctness.

```rust
//! Parse the fixed-size GGUF file header: magic + version + counts.

const GGUF_MAGIC: &[u8; 4] = b"GGUF";
const SUPPORTED_VERSION: u32 = 3;

#[derive(Debug, Clone, Copy)]
pub struct GgufHeader {
    pub version: u32,
    pub tensor_count: u64,
    pub metadata_count: u64,
}

/// Returns (header, bytes_consumed). Always 24 bytes for v3.
pub fn parse_header(bytes: &[u8]) -> anyhow::Result<(GgufHeader, usize)> {
    if bytes.len() < 24 {
        anyhow::bail!("header truncated: need 24 bytes, got {}", bytes.len());
    }
    if &bytes[0..4] != GGUF_MAGIC {
        anyhow::bail!("bad magic: expected b\"GGUF\", got {:?}", &bytes[0..4]);
    }
    let version = u32::from_le_bytes(bytes[4..8].try_into().unwrap());
    if version != SUPPORTED_VERSION {
        anyhow::bail!("unsupported GGUF version {version} (expected {SUPPORTED_VERSION})");
    }
    let tensor_count = u64::from_le_bytes(bytes[8..16].try_into().unwrap());
    let metadata_count = u64::from_le_bytes(bytes[16..24].try_into().unwrap());
    Ok((GgufHeader { version, tensor_count, metadata_count }, 24))
}
```

```rust
//! GGUF metadata: gguf_string and value-type dispatch.

#[derive(Debug, Clone)]
pub enum GgufValue {
    U8(u8),
    I8(i8),
    U16(u16),
    I16(i16),
    U32(u32),
    I32(i32),
    U64(u64),
    I64(i64),
    F32(f32),
    F64(f64),
    Bool(bool),
    String(String),
    Array(GgufArray),
}

#[derive(Debug, Clone)]
pub enum GgufArray {
    U32(Vec<u32>),
    U64(Vec<u64>),
    F32(Vec<f32>),
    String(Vec<String>),
    // Plan 7 only supports these array types; others reject.
}

/// Parse a length-prefixed UTF-8 string. Returns (value, bytes_consumed).
pub fn parse_string(bytes: &[u8]) -> anyhow::Result<(String, usize)> {
    if bytes.len() < 8 {
        anyhow::bail!("string length truncated");
    }
    let len = u64::from_le_bytes(bytes[0..8].try_into().unwrap()) as usize;
    if bytes.len() < 8 + len {
        anyhow::bail!("string truncated: declared len {len}, have {}", bytes.len() - 8);
    }
    let s = std::str::from_utf8(&bytes[8..8 + len])
        .map_err(|e| anyhow::anyhow!("invalid UTF-8 in gguf_string: {e}"))?
        .to_string();
    Ok((s, 8 + len))
}

/// Parse one (key, value, type) triple. Returns (key, value, bytes_consumed).
pub fn parse_kv(bytes: &[u8]) -> anyhow::Result<(String, GgufValue, usize)> {
    let (key, key_consumed) = parse_string(bytes)?;
    let rest = &bytes[key_consumed..];
    if rest.len() < 4 {
        anyhow::bail!("value type truncated");
    }
    let value_type = u32::from_le_bytes(rest[0..4].try_into().unwrap());
    let value_rest = &rest[4..];
    let (value, value_consumed) = parse_value(value_type, value_rest)?;
    Ok((key, value, key_consumed + 4 + value_consumed))
}

fn parse_value(type_id: u32, bytes: &[u8]) -> anyhow::Result<(GgufValue, usize)> {
    match type_id {
        0 => parse_primitive(bytes, 1).map(|(v, c)| (GgufValue::U8(v[0]), c)),
        1 => parse_primitive(bytes, 1).map(|(v, c)| (GgufValue::I8(v[0] as i8), c)),
        2 => {
            let buf = take(bytes, 2)?;
            Ok((GgufValue::U16(u16::from_le_bytes(buf.try_into().unwrap())), 2))
        }
        3 => {
            let buf = take(bytes, 2)?;
            Ok((GgufValue::I16(i16::from_le_bytes(buf.try_into().unwrap())), 2))
        }
        4 => {
            let buf = take(bytes, 4)?;
            Ok((GgufValue::U32(u32::from_le_bytes(buf.try_into().unwrap())), 4))
        }
        5 => {
            let buf = take(bytes, 4)?;
            Ok((GgufValue::I32(i32::from_le_bytes(buf.try_into().unwrap())), 4))
        }
        6 => {
            let buf = take(bytes, 4)?;
            Ok((GgufValue::F32(f32::from_le_bytes(buf.try_into().unwrap())), 4))
        }
        7 => parse_primitive(bytes, 1).map(|(v, c)| (GgufValue::Bool(v[0] != 0), c)),
        8 => {
            let (s, c) = parse_string(bytes)?;
            Ok((GgufValue::String(s), c))
        }
        9 => parse_array(bytes),
        10 => {
            let buf = take(bytes, 8)?;
            Ok((GgufValue::U64(u64::from_le_bytes(buf.try_into().unwrap())), 8))
        }
        11 => {
            let buf = take(bytes, 8)?;
            Ok((GgufValue::I64(i64::from_le_bytes(buf.try_into().unwrap())), 8))
        }
        12 => {
            let buf = take(bytes, 8)?;
            Ok((GgufValue::F64(f64::from_le_bytes(buf.try_into().unwrap())), 8))
        }
        other => anyhow::bail!("unknown GGUF value type {other}"),
    }
}

fn parse_primitive(bytes: &[u8], n: usize) -> anyhow::Result<(Vec<u8>, usize)> {
    if bytes.len() < n { anyhow::bail!("primitive truncated"); }
    Ok((bytes[0..n].to_vec(), n))
}

fn take<'a>(bytes: &'a [u8], n: usize) -> anyhow::Result<&'a [u8]> {
    if bytes.len() < n { anyhow::bail!("buffer truncated: need {n}, have {}", bytes.len()); }
    Ok(&bytes[..n])
}

fn parse_array(bytes: &[u8]) -> anyhow::Result<(GgufValue, usize)> {
    if bytes.len() < 12 { anyhow::bail!("array header truncated"); }
    let elem_type = u32::from_le_bytes(bytes[0..4].try_into().unwrap());
    let count = u64::from_le_bytes(bytes[4..12].try_into().unwrap()) as usize;
    let mut cursor = 12;
    match elem_type {
        4 => {
            let mut out = Vec::with_capacity(count);
            for _ in 0..count {
                let buf = take(&bytes[cursor..], 4)?;
                out.push(u32::from_le_bytes(buf.try_into().unwrap()));
                cursor += 4;
            }
            Ok((GgufValue::Array(GgufArray::U32(out)), cursor))
        }
        10 => {
            let mut out = Vec::with_capacity(count);
            for _ in 0..count {
                let buf = take(&bytes[cursor..], 8)?;
                out.push(u64::from_le_bytes(buf.try_into().unwrap()));
                cursor += 8;
            }
            Ok((GgufValue::Array(GgufArray::U64(out)), cursor))
        }
        6 => {
            let mut out = Vec::with_capacity(count);
            for _ in 0..count {
                let buf = take(&bytes[cursor..], 4)?;
                out.push(f32::from_le_bytes(buf.try_into().unwrap()));
                cursor += 4;
            }
            Ok((GgufValue::Array(GgufArray::F32(out)), cursor))
        }
        8 => {
            let mut out = Vec::with_capacity(count);
            for _ in 0..count {
                let (s, c) = parse_string(&bytes[cursor..])?;
                out.push(s);
                cursor += c;
            }
            Ok((GgufValue::Array(GgufArray::String(out)), cursor))
        }
        other => anyhow::bail!("unknown GGUF array element type {other}"),
    }
}
```

```rust
use super::metadata::parse_string;

#[derive(Debug, Clone, Copy, PartialEq, Eq)]
pub enum GgmlType {
    F32 = 0,
    F16 = 1,
    Q4_0 = 2,
    BF16 = 30,
}

#[derive(Debug, Clone)]
pub struct TensorDesc {
    pub name: String,
    pub shape: Vec<u64>,         // GGUF stores dims as u64; usually small enough for usize
    pub ggml_type: GgmlType,
    pub offset: u64,             // from start of tensor data section
}

pub fn parse_tensor_desc(bytes: &[u8]) -> anyhow::Result<(TensorDesc, usize)> {
    let (name, name_consumed) = parse_string(bytes)?;
    let mut cursor = name_consumed;

    if bytes.len() < cursor + 4 {
        anyhow::bail!("tensor desc truncated at n_dims");
    }
    let n_dims = u32::from_le_bytes(bytes[cursor..cursor+4].try_into().unwrap()) as usize;
    cursor += 4;

    if bytes.len() < cursor + n_dims * 8 {
        anyhow::bail!("tensor desc truncated at shape");
    }
    let mut shape = Vec::with_capacity(n_dims);
    for _ in 0..n_dims {
        shape.push(u64::from_le_bytes(bytes[cursor..cursor+8].try_into().unwrap()));
        cursor += 8;
    }

    if bytes.len() < cursor + 4 {
        anyhow::bail!("tensor desc truncated at ggml_type");
    }
    let type_id = u32::from_le_bytes(bytes[cursor..cursor+4].try_into().unwrap());
    let ggml_type = match type_id {
        0 => GgmlType::F32,
        1 => GgmlType::F16,
        2 => GgmlType::Q4_0,
        30 => GgmlType::BF16,
        other => anyhow::bail!("unsupported ggml_type {other} (only F32=0, F16=1, Q4_0=2, BF16=30 in Plan 7)"),
    };
    cursor += 4;

    if bytes.len() < cursor + 8 {
        anyhow::bail!("tensor desc truncated at offset");
    }
    let offset = u64::from_le_bytes(bytes[cursor..cursor+8].try_into().unwrap());
    cursor += 8;

    Ok((TensorDesc { name, shape, ggml_type, offset }, cursor))
}
```

```rust
use burn::tensor::{backend::Backend, Tensor, TensorData};
use half::f16;
use std::marker::PhantomData;

pub const Q4_0_BLOCK_SIZE: usize = 32;
pub const Q4_0_BYTES_PER_BLOCK: usize = 18;    // 2-byte scale + 16-byte nibbles

/// A weight tensor stored in GGUF's Q4_0 layout.
///
/// Storage:
///   - `blocks`: contiguous bytes; one 18-byte block per 32 values.
///   - `shape`: [in, out] math order. flat length must equal in × out and be a multiple of 32.
///
/// Flat layout convention: GGUF stores tensors in column-major over the SHAPE dims —
/// that is, for shape [d0, d1] the fastest-changing index is d0 in memory. The flat
/// buffer is `d1` "rows" of `d0` consecutive values each. Blocks chunk the flat buffer
/// into groups of 32.
///
/// When converting to a burn `Tensor<B, 2>` of shape `[in, out]`, we lay out the
/// dequantized values in burn's row-major convention: `result[i * out + j]` for input
/// row `i`, output column `j`. The GGUF flat order is column-major, so the mapping is:
///   gguf_flat_index(i, j) = j * in + i
pub struct Q4GgufTensor<B: Backend> {
    pub blocks: Vec<u8>,
    shape: [usize; 2],      // [in, out]
    _marker: PhantomData<B>,
    device: B::Device,
}

impl<B: Backend> Q4GgufTensor<B> {
    pub fn shape(&self) -> [usize; 2] { self.shape }

    pub fn from_blocks(blocks: Vec<u8>, shape: [usize; 2], device: &B::Device) -> anyhow::Result<Self> {
        let in_dim = shape[0];
        let out_dim = shape[1];
        let total = in_dim * out_dim;
        if total % Q4_0_BLOCK_SIZE != 0 {
            anyhow::bail!("Q4_0 requires total elements ({total}) divisible by {Q4_0_BLOCK_SIZE}");
        }
        let expected_bytes = (total / Q4_0_BLOCK_SIZE) * Q4_0_BYTES_PER_BLOCK;
        if blocks.len() != expected_bytes {
            anyhow::bail!("Q4_0 block bytes len mismatch: expected {expected_bytes}, got {}", blocks.len());
        }
        Ok(Self { blocks, shape, _marker: PhantomData, device: device.clone() })
    }

    /// Reconstruct an f32 Tensor<B, 2> of shape [in, out] in burn's row-major layout.
    pub fn dequantize(&self) -> Tensor<B, 2> {
        let in_dim = self.shape[0];
        let out_dim = self.shape[1];
        let total = in_dim * out_dim;
        let num_blocks = total / Q4_0_BLOCK_SIZE;

        // Reconstruct in GGUF's flat (column-major over [in, out]) order first.
        let mut gguf_flat = vec![0.0_f32; total];
        for b in 0..num_blocks {
            let off = b * Q4_0_BYTES_PER_BLOCK;
            let scale_bits = u16::from_le_bytes([self.blocks[off], self.blocks[off + 1]]);
            let scale = f16::from_bits(scale_bits).to_f32();
            for j in 0..16 {
                let byte = self.blocks[off + 2 + j];
                let low = (byte & 0x0F) as i32 - 8;
                let high = ((byte >> 4) & 0x0F) as i32 - 8;
                gguf_flat[b * Q4_0_BLOCK_SIZE + j]      = (low  as f32) * scale;
                gguf_flat[b * Q4_0_BLOCK_SIZE + j + 16] = (high as f32) * scale;
            }
        }

        // Transpose from column-major [in, out] to burn row-major [in, out]:
        //   burn_flat[i * out + j] = gguf_flat[j * in + i]
        let mut burn_flat = vec![0.0_f32; total];
        for i in 0..in_dim {
            for j in 0..out_dim {
                burn_flat[i * out_dim + j] = gguf_flat[j * in_dim + i];
            }
        }
        Tensor::<B, 2>::from_data(TensorData::new(burn_flat, [in_dim, out_dim]), &self.device)
    }
}
```

```rust
use crate::gguf::q4_0::Q4GgufTensor;
use crate::quant::{Q4Tensor, QuantizedTensor};
use burn::tensor::{backend::Backend, Tensor};

pub enum LinearWeight<B: Backend> {
    Dense(Tensor<B, 2>),
    Quantized(QuantizedTensor<B>),
    Q4(Q4Tensor<B>),
    Q4Gguf(Q4GgufTensor<B>),
}

impl<B: Backend> LinearWeight<B> {
    pub fn shape(&self) -> [usize; 2] {
        match self {
            Self::Dense(t) => t.dims(),
            Self::Quantized(q) => q.shape(),
            Self::Q4(q) => q.shape(),
            Self::Q4Gguf(q) => q.shape(),
        }
    }

    pub fn matmul(&self, x: Tensor<B, 3>) -> Tensor<B, 3> {
        match self {
            Self::Dense(w) => x.matmul(w.clone().unsqueeze()),
            Self::Quantized(q) => x.matmul(q.dequantize().unsqueeze()),
            Self::Q4(q) => x.matmul(q.dequantize().unsqueeze()),
            Self::Q4Gguf(q) => x.matmul(q.dequantize().unsqueeze()),
        }
    }

    pub fn swap_dims(self, a: usize, b: usize) -> Self {
        match self {
            Self::Dense(t) => Self::Dense(t.swap_dims(a, b)),
            Self::Quantized(q) => Self::Quantized(q.transpose_2d()),
            Self::Q4(q) => Self::Q4(Q4Tensor::quantize_from(q.dequantize().swap_dims(a, b))),
            Self::Q4Gguf(_) => {
                // GGUF Q4_0 weights are always loaded in math order. swap_dims
                // is conceptually wrong here — callers should use
                // `ensure_math_order` instead, which is a no-op for this variant.
                // If we DO get called, force-dequantize + swap + RE-encode would
                // be lossy; we panic to surface the misuse.
                panic!("swap_dims called on Q4Gguf — use ensure_math_order instead");
            }
        }
    }

    pub fn ensure_math_order(self) -> Self {
        match self {
            Self::Q4(_) | Self::Q4Gguf(_) => self,
            _ => self.swap_dims(0, 1),
        }
    }
}
```

```rust
use crate::gguf::header::parse_header;
use crate::gguf::metadata::parse_kv;
use crate::gguf::tensor_desc::{parse_tensor_desc, GgmlType, TensorDesc};
use crate::gguf::q4_0::Q4GgufTensor;
use crate::name_map::hf_from_gguf;
use std::collections::HashMap;

const TENSOR_DATA_ALIGN: u64 = 32;

/// Load a GGUF file. Returns a LoadedWeights for the layer range.
pub fn load_gguf<B: Backend>(
    path: &std::path::Path,
    cfg: &crate::config::ModelConfig,
    layer_range: std::ops::Range<usize>,
    hosts_embedding: bool,
    hosts_output: bool,
    device: &B::Device,
) -> anyhow::Result<LoadedWeights<B>> {
    let file = std::fs::File::open(path)
        .with_context(|| format!("open {}", path.display()))?;
    let mmap = unsafe { memmap2::Mmap::map(&file) }
        .with_context(|| format!("mmap {}", path.display()))?;

    // 1. Header
    let (hdr, mut cursor) = parse_header(&mmap)?;

    // 2. Metadata (consume `metadata_count` KV pairs)
    let mut metadata: HashMap<String, crate::gguf::metadata::GgufValue> = HashMap::new();
    for _ in 0..hdr.metadata_count {
        let (k, v, consumed) = parse_kv(&mmap[cursor..])?;
        metadata.insert(k, v);
        cursor += consumed;
    }

    // 3. Tensor descriptors
    let mut descs: Vec<TensorDesc> = Vec::with_capacity(hdr.tensor_count as usize);
    for _ in 0..hdr.tensor_count {
        let (d, consumed) = parse_tensor_desc(&mmap[cursor..])?;
        descs.push(d);
        cursor += consumed;
    }

    // 4. Align cursor to tensor data section start
    let alignment = TENSOR_DATA_ALIGN;    // could read from `general.alignment` metadata
    cursor = ((cursor as u64 + alignment - 1) & !(alignment - 1)) as usize;
    let data_start = cursor;

    // 5. Build a lookup of {hf_name → (TensorDesc, slice)}
    let mut by_hf: HashMap<String, (TensorDesc, &[u8])> = HashMap::new();
    for d in &descs {
        let Some(hf_name) = hf_from_gguf(&d.name) else { continue; };
        let off = data_start + d.offset as usize;
        let size = tensor_bytes(d)?;
        if off + size > mmap.len() {
            anyhow::bail!("tensor `{}` extends past file end", d.name);
        }
        by_hf.insert(hf_name, (d.clone(), &mmap[off..off + size]));
    }

    // 6. Build LayerWeights via load helpers.
    let load_2d = |hf_name: &str| -> anyhow::Result<Tensor<B, 2>> {
        let (d, slice) = by_hf.get(hf_name)
            .ok_or_else(|| anyhow::anyhow!("missing tensor `{hf_name}` in GGUF"))?;
        load_dense_2d::<B>(d, slice, device)
    };
    let load_1d = |hf_name: &str| -> anyhow::Result<Tensor<B, 1>> {
        let (d, slice) = by_hf.get(hf_name)
            .ok_or_else(|| anyhow::anyhow!("missing tensor `{hf_name}` in GGUF"))?;
        load_dense_1d::<B>(d, slice, device)
    };
    let load_lin = |hf_name: &str| -> anyhow::Result<LinearWeight<B>> {
        let (d, slice) = by_hf.get(hf_name)
            .ok_or_else(|| anyhow::anyhow!("missing tensor `{hf_name}` in GGUF"))?;
        match d.ggml_type {
            GgmlType::Q4_0 => {
                let shape = [d.shape[0] as usize, d.shape[1] as usize];
                Ok(LinearWeight::Q4Gguf(Q4GgufTensor::from_blocks(slice.to_vec(), shape, device)?))
            }
            _ => {
                let t = load_dense_2d::<B>(d, slice, device)?;
                Ok(LinearWeight::Dense(t))
            }
        }
    };

    let embedding = if hosts_embedding {
        Some(load_2d("model.embed_tokens.weight").or_else(|_| {
            // Tied embedding: GGUF stores only `output.weight` (which becomes lm_head).
            // Dequantize that into a Dense embedding.
            let (d, slice) = by_hf.get("lm_head.weight")
                .ok_or_else(|| anyhow::anyhow!("neither model.embed_tokens.weight nor lm_head.weight found"))?;
            match d.ggml_type {
                GgmlType::Q4_0 => {
                    let shape = [d.shape[0] as usize, d.shape[1] as usize];
                    let lw = LinearWeight::Q4Gguf(Q4GgufTensor::from_blocks(slice.to_vec(), shape, device)?);
                    // lw is [hidden, vocab] (GGUF math order); embedding needs [vocab, hidden].
                    let dense = match lw {
                        LinearWeight::Q4Gguf(q) => q.dequantize(),
                        _ => unreachable!(),
                    };
                    Ok(dense.swap_dims(0, 1))
                }
                _ => load_dense_2d::<B>(d, slice, device).map(|t| t.swap_dims(0, 1)),
            }
        })?)
    } else { None };

    let mut layers = Vec::with_capacity(layer_range.len());
    for i in layer_range.clone() {
        layers.push(LayerWeights {
            attn_norm: load_1d(&format!("model.layers.{i}.input_layernorm.weight"))?,
            q_proj:    load_lin(&format!("model.layers.{i}.self_attn.q_proj.weight"))?,
            k_proj:    load_lin(&format!("model.layers.{i}.self_attn.k_proj.weight"))?,
            v_proj:    load_lin(&format!("model.layers.{i}.self_attn.v_proj.weight"))?,
            o_proj:    load_lin(&format!("model.layers.{i}.self_attn.o_proj.weight"))?,
            ffn_norm:  load_1d(&format!("model.layers.{i}.post_attention_layernorm.weight"))?,
            ffn_gate:  load_lin(&format!("model.layers.{i}.mlp.gate_proj.weight"))?,
            ffn_up:    load_lin(&format!("model.layers.{i}.mlp.up_proj.weight"))?,
            ffn_down:  load_lin(&format!("model.layers.{i}.mlp.down_proj.weight"))?,
        });
    }

    let final_norm = if hosts_output {
        Some(load_1d("model.norm.weight")?)
    } else { None };

    let output_proj = if hosts_output && !cfg.tie_word_embeddings {
        Some(load_lin("lm_head.weight")?)
    } else { None };

    let _ = metadata;   // reserved for future use (e.g., chat template, vocab)
    Ok(LoadedWeights { embedding, layers, final_norm, output_proj })
}

fn tensor_bytes(d: &TensorDesc) -> anyhow::Result<usize> {
    let total: u64 = d.shape.iter().product();
    match d.ggml_type {
        GgmlType::F32 => Ok((total * 4) as usize),
        GgmlType::F16 | GgmlType::BF16 => Ok((total * 2) as usize),
        GgmlType::Q4_0 => {
            if total % crate::gguf::q4_0::Q4_0_BLOCK_SIZE as u64 != 0 {
                anyhow::bail!("Q4_0 tensor `{}` not multiple of 32 elements", d.name);
            }
            Ok((total / crate::gguf::q4_0::Q4_0_BLOCK_SIZE as u64 * crate::gguf::q4_0::Q4_0_BYTES_PER_BLOCK as u64) as usize)
        }
    }
}

fn load_dense_2d<B: Backend>(d: &TensorDesc, bytes: &[u8], device: &B::Device) -> anyhow::Result<Tensor<B, 2>> {
    if d.shape.len() != 2 {
        anyhow::bail!("`{}` expected 2D, got {} dims", d.name, d.shape.len());
    }
    // GGUF math order [in, out] = [shape[0], shape[1]]. Transpose to burn's row-major
    // [in, out] layout requires re-laying out: GGUF flat is column-major over [in, out];
    // burn flat is row-major. Reuse the convert logic from Q4Gguf::dequantize via a free fn.
    let in_dim = d.shape[0] as usize;
    let out_dim = d.shape[1] as usize;
    let total = in_dim * out_dim;
    let f32_data = bytes_to_f32_vec_gguf(bytes, d.ggml_type, total)?;
    // f32_data is in GGUF column-major; transpose to row-major.
    let mut burn_flat = vec![0.0_f32; total];
    for i in 0..in_dim {
        for j in 0..out_dim {
            burn_flat[i * out_dim + j] = f32_data[j * in_dim + i];
        }
    }
    Ok(Tensor::<B, 2>::from_data(TensorData::new(burn_flat, [in_dim, out_dim]), device))
}

fn load_dense_1d<B: Backend>(d: &TensorDesc, bytes: &[u8], device: &B::Device) -> anyhow::Result<Tensor<B, 1>> {
    if d.shape.len() != 1 {
        anyhow::bail!("`{}` expected 1D, got {} dims", d.name, d.shape.len());
    }
    let total = d.shape[0] as usize;
    let f32_data = bytes_to_f32_vec_gguf(bytes, d.ggml_type, total)?;
    Ok(Tensor::<B, 1>::from_data(TensorData::new(f32_data, [total]), device))
}

fn bytes_to_f32_vec_gguf(bytes: &[u8], ggml_type: GgmlType, expected_elements: usize) -> anyhow::Result<Vec<f32>> {
    match ggml_type {
        GgmlType::F32 => {
            let v: &[f32] = bytemuck::cast_slice(bytes);
            if v.len() != expected_elements {
                anyhow::bail!("F32 element count mismatch: expected {expected_elements}, got {}", v.len());
            }
            Ok(v.to_vec())
        }
        GgmlType::F16 => {
            let v: &[u16] = bytemuck::cast_slice(bytes);
            Ok(v.iter().map(|b| half::f16::from_bits(*b).to_f32()).collect())
        }
        GgmlType::BF16 => {
            let v: &[u16] = bytemuck::cast_slice(bytes);
            Ok(v.iter().map(|b| half::bf16::from_bits(*b).to_f32()).collect())
        }
        GgmlType::Q4_0 => anyhow::bail!("Q4_0 cannot be loaded as dense; use Q4Gguf path"),
    }
}
```

```python
import numpy as np
import torch
from safetensors.torch import load_file

OUT = Path(__file__).resolve().parent.parent / "fixtures" / "toy-llama-3-gguf"
OUT.mkdir(parents=True, exist_ok=True)

# --- GGUF constants ---
GGUF_MAGIC = b"GGUF"
GGUF_VERSION = 3
TYPE_U8, TYPE_I8, TYPE_U16, TYPE_I16, TYPE_U32, TYPE_I32 = range(6)
TYPE_F32, TYPE_BOOL, TYPE_STRING, TYPE_ARRAY = 6, 7, 8, 9
TYPE_U64, TYPE_I64, TYPE_F64 = 10, 11, 12
GGML_F32, GGML_F16, GGML_Q4_0, GGML_BF16 = 0, 1, 2, 30

Q4_0_BLOCK = 32
Q4_0_BYTES_PER_BLOCK = 18

LINEAR_PATTERNS = [
    "self_attn.q_proj.weight",
    "self_attn.k_proj.weight",
    "self_attn.v_proj.weight",
    "self_attn.o_proj.weight",
    "mlp.gate_proj.weight",
    "mlp.up_proj.weight",
    "mlp.down_proj.weight",
    "lm_head.weight",
]

HF_TO_GGUF = {
    "model.embed_tokens.weight": "token_embd.weight",
    "model.norm.weight": "output_norm.weight",
    "lm_head.weight": "output.weight",
}
def hf_to_gguf_name(name: str) -> str:
    if name in HF_TO_GGUF:
        return HF_TO_GGUF[name]
    if name.startswith("model.layers."):
        parts = name.split(".")
        # model.layers.N.<rest>
        i = parts[2]
        rest = ".".join(parts[3:])
        return {
            "input_layernorm.weight": f"blk.{i}.attn_norm.weight",
            "self_attn.q_proj.weight": f"blk.{i}.attn_q.weight",
            "self_attn.k_proj.weight": f"blk.{i}.attn_k.weight",
            "self_attn.v_proj.weight": f"blk.{i}.attn_v.weight",
            "self_attn.o_proj.weight": f"blk.{i}.attn_output.weight",
            "post_attention_layernorm.weight": f"blk.{i}.ffn_norm.weight",
            "mlp.gate_proj.weight": f"blk.{i}.ffn_gate.weight",
            "mlp.up_proj.weight": f"blk.{i}.ffn_up.weight",
            "mlp.down_proj.weight": f"blk.{i}.ffn_down.weight",
        }[rest]
    raise ValueError(f"unhandled tensor name: {name}")

def quantize_q4_0(w: np.ndarray) -> bytes:
    """Quantize a 1D f32 array (length multiple of 32) into Q4_0 blocks.
    Block layout: f16 scale + 16 bytes of nibbles, with low nibbles = indices 0..16, high = 16..32.
    """
    assert w.dtype == np.float32 and w.size % Q4_0_BLOCK == 0
    n_blocks = w.size // Q4_0_BLOCK
    out = bytearray()
    for b in range(n_blocks):
        block = w[b * Q4_0_BLOCK:(b + 1) * Q4_0_BLOCK]
        max_abs = float(np.max(np.abs(block)))
        if max_abs == 0.0:
            scale = 1.0
            quantized = np.zeros(Q4_0_BLOCK, dtype=np.int32)
        else:
            scale = max_abs / 8.0  # GGUF uses signed range -8..7, so divide by 8
            quantized = np.clip(np.round(block / scale).astype(np.int32), -8, 7)
        # f16 scale
        out.extend(np.float16(scale).tobytes())
        # 16 bytes of nibbles
        # qs[j] (j in 0..16): low nibble = block[j], high nibble = block[j + 16]
        for j in range(16):
            lo = (quantized[j] + 8) & 0x0F
            hi = (quantized[j + 16] + 8) & 0x0F
            out.append((hi << 4) | lo)
    return bytes(out)

def write_gguf_string(buf: bytearray, s: str):
    data = s.encode("utf-8")
    buf.extend(struct.pack("<Q", len(data)))
    buf.extend(data)

def write_kv_u32(buf, key, value):
    write_gguf_string(buf, key)
    buf.extend(struct.pack("<I", TYPE_U32))
    buf.extend(struct.pack("<I", value))

def write_kv_u64(buf, key, value):
    write_gguf_string(buf, key)
    buf.extend(struct.pack("<I", TYPE_U64))
    buf.extend(struct.pack("<Q", value))

def write_kv_f32(buf, key, value):
    write_gguf_string(buf, key)
    buf.extend(struct.pack("<I", TYPE_F32))
    buf.extend(struct.pack("<f", value))

def write_kv_string(buf, key, value):
    write_gguf_string(buf, key)
    buf.extend(struct.pack("<I", TYPE_STRING))
    write_gguf_string(buf, value)

# --- Build GGUF file ---

# Load bf16 source.
src = load_file(SRC / "model.safetensors")
cfg = json.loads((SRC / "config.json").read_text())

# Metadata
meta = bytearray()
meta_count = 0
def meta_add(fn):
    nonlocal_count = [meta_count]
    return fn

write_kv_string(meta, "general.architecture", "llama"); meta_count += 1
write_kv_string(meta, "general.name", "toy-llama-3-q4-0"); meta_count += 1
write_kv_u32(meta, "general.alignment", 32); meta_count += 1
write_kv_u32(meta, "llama.block_count", cfg["num_hidden_layers"]); meta_count += 1
write_kv_u32(meta, "llama.embedding_length", cfg["hidden_size"]); meta_count += 1
write_kv_u32(meta, "llama.attention.head_count", cfg["num_attention_heads"]); meta_count += 1
write_kv_u32(meta, "llama.attention.head_count_kv", cfg["num_key_value_heads"]); meta_count += 1
write_kv_u32(meta, "llama.feed_forward_length", cfg["intermediate_size"]); meta_count += 1
write_kv_u32(meta, "llama.context_length", cfg["max_position_embeddings"]); meta_count += 1
write_kv_f32(meta, "llama.attention.layer_norm_rms_epsilon", cfg["rms_norm_eps"]); meta_count += 1
write_kv_f32(meta, "llama.rope.freq_base", cfg["rope_theta"]); meta_count += 1

# Build tensors
tensor_descs = bytearray()
tensor_data = bytearray()
tensor_count = 0

def write_tensor_desc(name, n_dims, shape, ggml_type, offset):
    write_gguf_string(tensor_descs, name)
    tensor_descs.extend(struct.pack("<I", n_dims))
    for d in shape:
        tensor_descs.extend(struct.pack("<Q", d))
    tensor_descs.extend(struct.pack("<I", ggml_type))
    tensor_descs.extend(struct.pack("<Q", offset))

def should_quantize(name: str) -> bool:
    return any(p in name for p in LINEAR_PATTERNS)

ALIGN = 32

def append_data_with_alignment(data: bytes) -> int:
    # Align to 32 bytes within tensor_data buffer.
    while len(tensor_data) % ALIGN != 0:
        tensor_data.append(0)
    off = len(tensor_data)
    tensor_data.extend(data)
    return off

for hf_name, t in src.items():
    gguf_name = hf_to_gguf_name(hf_name)
    arr = t.to(torch.float32).numpy()
    if should_quantize(hf_name):
        # Transpose HF [out, in] -> GGUF [in, out] (i.e., math order, also column-major flat = same as transposing).
        # In numpy, .T flips axes; the underlying flat memory becomes Fortran-order [in, out] which matches GGUF.
        arr_T = arr.T.copy()                          # [in, out] C-order
        flat_col_major = arr_T.flatten(order="F")      # [in, out] flattened column-major = same data as C-order [out, in] flattened C
        # Actually for GGUF "column-major over [in, out]" we want indices to go (in0_out0, in1_out0, ..., in31_out0, in0_out1, ...)
        # Which is exactly `arr.T.flatten(order="F")` — same as `arr.flatten(order="C")` if shapes are [out, in].
        # Cross-check: arr has shape [out, in], arr.flatten(C-order) gives out0_in0, out0_in1, ..., out0_in_last, out1_in0, ...
        # That's (out outer, in inner) = column-major of [in, out] if we relabel.
        # So just use arr.flatten() directly without transposing.
        flat = arr.flatten().astype(np.float32)
        q = quantize_q4_0(flat)
        offset = append_data_with_alignment(q)
        # GGUF tensor shape is [in, out] (math order).
        write_tensor_desc(gguf_name, 2, [arr.shape[1], arr.shape[0]], GGML_Q4_0, offset)
    else:
        # F32 for non-linear tensors.
        if arr.ndim == 1:
            data = arr.astype(np.float32).tobytes()
            offset = append_data_with_alignment(data)
            write_tensor_desc(gguf_name, 1, [arr.shape[0]], GGML_F32, offset)
        elif arr.ndim == 2:
            # Same flat-layout reasoning as above.
            data = arr.astype(np.float32).flatten().tobytes()
            offset = append_data_with_alignment(data)
            write_tensor_desc(gguf_name, 2, [arr.shape[1], arr.shape[0]], GGML_F32, offset)
        else:
            raise ValueError(f"unhandled tensor rank: {arr.shape}")
    tensor_count += 1

# Assemble final file.
header = bytearray()
header.extend(GGUF_MAGIC)
header.extend(struct.pack("<I", GGUF_VERSION))
header.extend(struct.pack("<Q", tensor_count))
header.extend(struct.pack("<Q", meta_count))

# Pad after tensor descriptors to ALIGN before tensor data.
total_before_data = len(header) + len(meta) + len(tensor_descs)
pad = (ALIGN - (total_before_data % ALIGN)) % ALIGN

with open(OUT / "model.gguf", "wb") as f:
    f.write(header)
    f.write(meta)
    f.write(tensor_descs)
    f.write(b"\x00" * pad)
    f.write(tensor_data)

# Copy reference logits + tokenizer + reference prompt + config.
shutil.copy(SRC / "reference_logits.bin", OUT / "reference_logits.bin")
shutil.copy(SRC / "tokenizer.json", OUT / "tokenizer.json")
shutil.copy(SRC / "reference_prompt.txt", OUT / "reference_prompt.txt")
shutil.copy(SRC / "config.json", OUT / "config.json")

(OUT / "README.md").write_text(

GGUF Q4_0 layout: per-32 block, f16 scale + 16 bytes of biased nibbles (low half = indices 0..16, high half = 16..32).
""")

# Stats
bf16_size = sum(t.numel() * t.element_size() for t in src.values())
gguf_size = (OUT / "model.gguf").stat().st_size
print(f"bf16 fixture:  {bf16_size} bytes")
print(f"GGUF fixture:  {gguf_size} bytes")
print(f"compression:   {bf16_size / gguf_size:.2f}x")
```

```rust
    let prompt = std::fs::read_to_string(fix.join("reference_prompt.txt")).unwrap();
    let ids = tok.encode(prompt.trim()).unwrap();

    let dev = Default::default();
    let weights = load_gguf::<B>(
        &fix.join("model.gguf"), &cfg,
        0..cfg.n_layers, true, true, &dev,
    ).unwrap();
    let model = Model::<B>::from_loaded(&cfg, weights, &dev).unwrap();

    let ids_i32: Vec<i32> = ids.iter().map(|x| *x as i32).collect();
    let token_ids = Tensor::<B, 2, Int>::from_data(
        TensorData::new(ids_i32, [1, ids.len()]),
        &dev,
    );
    let logits = model.forward(token_ids, 0);
    let last_pos_logits: Tensor<B, 1> = logits
        .slice([0..1, (ids.len() - 1)..ids.len(), 0..cfg.vocab_size])
        .reshape([cfg.vocab_size]);

    let got: Vec<f32> = last_pos_logits.to_data().to_vec().unwrap();
    assert_eq!(got.len(), cfg.vocab_size);
    // Finite-value check: no NaN, no Inf.
    for (i, v) in got.iter().enumerate() {
        assert!(v.is_finite(), "logit[{i}] is not finite: {v}");
    }

    // Diff against bf16 reference (informational; do not assert tight tolerance
    // on the random-init toy — GGUF Q4_0 + GGUF flat-layout transpose introduces
    // significant intrinsic error that's not a bug). Just assert max_diff is bounded.
    let ref_bytes = std::fs::read(fix.join("reference_logits.bin")).unwrap();
    let ref_f32: &[f32] = bytemuck::cast_slice(&ref_bytes);
    let mut max_diff = 0.0_f32;
    for (a, b) in got.iter().zip(ref_f32.iter()) {
        let d = (a - b).abs();
        if d > max_diff { max_diff = d; }
    }
    eprintln!("GGUF Q4_0 vs bf16 reference max |a-b| = {max_diff}");
    // Sanity bound: random-weight GGUF Q4_0 forward must produce SOME bounded
    // output. If it's > 10 (i.e., logits diverge wildly), there's a real bug.
    assert!(max_diff < 10.0, "GGUF Q4_0 logits diverged catastrophically: {max_diff}");
}
```

1. Tensor data offset alignment wrong — verify the cursor math in `load_gguf` aligns to 32 after tensor descriptors.
2. GGUF shape order — the script writes `[shape[1], shape[0]]` (i.e., [in, out]) for transposed tensors; the loader reads `[d.shape[0], d.shape[1]]` which should be `[in, out]`. Verify they agree.
3. Q4_0 block decode — bias is `-8` (we map nibble 0..15 to signed -8..7), not `-7`.
4. Column-major→row-major transpose direction — `burn_flat[i * out + j] = gguf_flat[j * in + i]`. Off-by-one or transpose-flipped is a common error.

**Acknowledged risks:**

1. **GGUF column-major flat layout is THE bug-prone bit.** GGUF tensors are stored with the first dim as the contiguous (fastest-changing) dim — i.e., for `[in, out]` the flat order is `(in=0, out=0), (in=1, out=0), ..., (in=last, out=0), (in=0, out=1), ...`. To get burn's row-major `[in, out]` layout (`(in=i, out=j)` at `i * out + j`), we apply the transpose `burn_flat[i * out + j] = gguf_flat[j * in + i]`. If we get this backwards, the forward pass produces garbage.

2. **Q4_0 nibble bias** is `-8` (signed range -8..7), NOT `-7` like our own Q4 format. Confusing the two = wrong values.

3. **Q4_0 half-split** is `qs[j]` low-nibble = weight[j], high-nibble = weight[j+16]. NOT alternating columns like our own format. Confusing them = scrambled weights.

4. **Tensor data alignment**: after the variable-length descriptor section, the file is padded to `general.alignment` (default 32) before tensor data starts. Mismatch = read wrong byte offsets.

5. **Random toy weights make the reference-logits gate informational, not strict.** The dispatch-parity test from Plan 6 still proves the dequantize-then-matmul path is correct; this plan's gate just confirms the loader plumbing doesn't blow up. For real-model validation, see future plans.
